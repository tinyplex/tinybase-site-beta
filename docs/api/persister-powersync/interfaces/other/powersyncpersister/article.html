PowerSyncPersister

The PowerSyncPersister interface is a minor extension to the Persister interface.

It simply provides an extra getPowerSync method for accessing a reference to the PowerSync instance the Store is being persisted to.

You should use the createPowerSyncPersister function to create a PowerSyncPersister object.

Since

v4.8.0

getPowerSync

The getPowerSync method returns a reference to the PowerSync instance the Store is being persisted to.
