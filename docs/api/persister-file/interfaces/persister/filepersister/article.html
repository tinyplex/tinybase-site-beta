FilePersister

The FilePersister interface is a minor extension to the Persister interface.

It simply provides an extra getFilePath method for accessing the location of the local file the Store is being persisted to.

You should use the createFilePersister function to create a FilePersister object.

Since

v4.3.14

getFilePath

The getFilePath method returns the location of the local file the Store is being persisted to.
