getClientIds

The getClientIds method is used to access a list of all the connected clients on a given path.

getClientIds(pathId: string): Ids
TypeDescription
pathIdstring
returnsIds

Note that if you call this method from the onClientId method as a client is getting removed, it will still be returned in the list of client Ids.

Since

v5.4.0
