Modelling methods

This is the collection of modelling methods within the Tools interface. There are only three modelling methods, getPrettyStoreApi, getStoreApi, and getStoreSchema.

getPrettyStoreApi

The getStoreApi method attempts to return a prettified code-generated .d.ts file and a .ts file that wraps the schema of a Store.

getStoreApi

The getStoreApi method returns a code-generated .d.ts file and a .ts file that wraps the schema of a Store.

getStoreSchema

The getStoreSchema method returns the Schema of the Store as an object.
