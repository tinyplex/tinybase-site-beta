Internal type aliases

This is the collection of internal type aliases within the store module. There are only two internal type aliases, TablesFrom and CellFrom.

TablesFrom

The TablesFrom type is a utility for determining a Tables structure from a provided Schema.

CellFrom

The CellFrom type is a utility for determining a Cell type from a provided Schema.
