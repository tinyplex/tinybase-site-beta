Internal type aliases

This is the collection of internal type aliases within the store module. There are 5 internal type aliases in total.

TablesFromSchema

The TablesFromSchema type is a utility for determining a Tables structure from a provided Schema.

NoTablesSchema

The NoTablesSchema type is a TablesSchema-like type for when one has not been provided.

CellIdsDefaultedFromSchema

The CellIdsDefaultedFromSchema type is a utility for determining Cells with or without default status from a provided Schema.

CellTypeFromSchema

The CellTypeFromSchema type is a utility for determining a Cell type from a provided Schema.

NoValuesSchema

The NoValuesSchema type is a ValuesSchema-like type for when one has not been provided.
