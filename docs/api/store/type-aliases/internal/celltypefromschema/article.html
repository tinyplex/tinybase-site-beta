CellTypeFromSchema

The CellTypeFromSchema type is a utility for determining a Cell type from a provided Schema.

CellType extends "string" ? string : CellType extends "number" ? number : CellType extends "boolean" ? boolean : Cell

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
