CellIdsDefaultedFromSchema

The CellIdsDefaultedFromSchema type is a utility for determining Cells with or without default status from a provided Schema.

{[CellId in keyof TableSchema]: TableSchema[CellId] extends {default: Cell} ? IsDefaulted extends true ? CellId : never : IsDefaulted extends true ? never : CellId}[keyof TableSchema]

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
