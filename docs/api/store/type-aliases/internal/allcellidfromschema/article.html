AllCellIdFromSchema

The AllCellIdFromSchema type is a utility for determining the Id of Cells across all Tables, from a provided TablesSchema.

TableId extends TableIdFromSchema<Schema> ? CellIdFromSchema<Schema, TableId> : never

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
