ValueFromSchema

The ValueTypeFromSchema type is a utility for determining a Value type from a provided ValuesSchema.

ValueType extends "string" ? string : ValueType extends "number" ? number : ValueType extends "boolean" ? boolean : string | number | boolean

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
