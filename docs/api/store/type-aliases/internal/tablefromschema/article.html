TableFromSchema

The TableFromSchema type is a utility for determining a Table structure from a provided TablesSchema.

{[rowId: Id]: RowFromSchema<Schema, TableId, WhenSet>}

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
