TablesFrom

The TablesFrom type is a utility for determining a Tables structure from a provided Schema.

{-readonly [TableId in keyof Schema]-?: {[rowId: Id]: {-readonly [CellId in keyof Schema[TableId]]-?: CellFrom<Schema, TableId, CellId>}}}

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
