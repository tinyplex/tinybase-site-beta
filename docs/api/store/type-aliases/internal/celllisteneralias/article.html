CellListenerAlias

The CellListenerAlias type is a duplicate of CellListener, used to mask complex generics from documentation.

CellListener<Schemas, TableIdOrNull, RowIdOrNull, CellIdOrNull>

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
