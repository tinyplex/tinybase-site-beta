NoTablesSchema

The NoTablesSchema type is a TablesSchema-like type for when one has not been provided.

{[tableId: Id]: {[cellId: Id]: {type: "any"}}}

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
