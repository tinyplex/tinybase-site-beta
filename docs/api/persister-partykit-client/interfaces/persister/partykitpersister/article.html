PartyKitPersister

The PartyKitPersister interface is a minor extension to the Persister interface.

It simply provides an extra getConnection method for accessing the PartySocket the Store is being persisted to.

You should use the createPartyKitPersister function to create a PartyKitPersister object.

Since

v4.3.14

getConnection

The getConnection method returns the PartySocket the Store is being persisted to.
