YjsPersister

The YjsPersister interface is a minor extension to the Persister interface.

It simply provides an extra getYDoc method for accessing the Yjs document the Store is being persisted to.

You should use the createYjsPersister function to create a YjsPersister object.

Since

v4.3.14

getYDoc

The getYDoc method returns the Yjs document the Store is being persisted to.
