JoinedCellIdOrId

The JoinedCellIdOrId type is a utility for returning a schema-specific Id (if the Table is known), or a plain Id (if it is merely an alias for a Table).

JoinedTableId extends TableIdFromSchema<Schema> ? CellIdFromSchema<Schema, JoinedTableId> : Id

This type is used internally to the TinyBase type system and you are not expected to need to use it directly.
