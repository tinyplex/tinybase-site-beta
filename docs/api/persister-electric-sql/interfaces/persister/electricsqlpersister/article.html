ElectricSqlPersister

The ElectricSqlPersister interface is a minor extension to the Persister interface.

It simply provides an extra getElectricClient method for accessing a reference to the Electric client the Store is being persisted to.

You should use the createElectricSqlPersister function to create an ElectricSqlPersister object.

Since

v4.6.0

getElectricClient

The getElectricClient method returns a reference to the Electric client the Store is being persisted to.
