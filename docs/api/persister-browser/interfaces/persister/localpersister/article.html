LocalPersister

The LocalPersister interface is a minor extension to the Persister interface.

It simply provides an extra getStorageName method for accessing the unique key of the storage location the Store is being persisted to.

You should use the createLocalPersister function to create a LocalPersister object.

Since

v4.3.14

getStorageName

The getStorageName method returns the unique key of the storage location the Store is being persisted to.
