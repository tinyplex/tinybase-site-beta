ExpoSqlitePersister

The ExpoSqlitePersister interface is a minor extension to the Persister interface.

It simply provides an extra getDb method for accessing a reference to the database instance the Store is being persisted to.

You should use the createExpoSqlitePersister function to create an ExpoSqlitePersister object.

Since

v4.3.14

getDb

The getDb method returns a reference to the database instance the Store is being persisted to.
